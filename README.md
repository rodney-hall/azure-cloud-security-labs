# Azure Cloud Security Labs

This repository contains hands-on Azure cloud security labs built to demonstrate practical security concepts aligned with entry-level cloud and cybersecurity roles.

The labs focus on identity and access management, least privilege, and secure Azure resource configuration using Microsoft Azure.

## Skills Demonstrated
- Azure Role-Based Access Control (RBAC)
- Microsoft Entra ID (Azure AD)
- Least Privilege Access
- Azure IAM (Identity and Access Management)
- Cloud Security Fundamentals
